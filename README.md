# Nomade - Clon Airbnb

Proyecto de interfaz tipo Airbnb construido con Next.js 16, TypeScript, Tailwind CSS y App Router.

## Tecnologias

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4

## Rutas

- /
- /catalog
- /rooms/[id]

## Ejecucion local

```bash
npm install
npm run dev
```

Abrir http://localhost:3000.

## Build de produccion

```bash
npm run build
npm run start
```
